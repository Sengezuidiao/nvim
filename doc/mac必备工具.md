# 包管理器`homebrew`

* `brew list` 查看已经安装的软件
  * 其中`Formulae` 是常规命令行程序
  * `Casks`为图形化程序。
* 安装应用程序
  * 搜索程序, `brew searce 可能包名`
  * 安装命令行程序, `brew install 包名`
  * 安装图形化程序, `brew --cask install 包名`
